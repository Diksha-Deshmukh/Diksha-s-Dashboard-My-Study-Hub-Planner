# Study Dashboard – My Study Hub & Planner

A modern, intelligent study planner application built with React and Vite. Track your study hours, manage tasks, schedule sessions, and visualize your progress all in one place.

## Features

- 📚 **Task Management** - Create, organize, and track your study tasks
- 📅 **Study Scheduling** - Plan your study sessions with an intuitive scheduler
- 📊 **Progress Tracking** - Visualize study progress with interactive charts
- ⏱️ **Study Hours Logging** - Record and track daily study hours
- 🎨 **Modern UI** - Clean, responsive interface built with Tailwind CSS
- ⚡ **Lightning Fast** - Built with Vite for instant hot module replacement

## Tech Stack

- **Frontend Framework:** React 18
- **Build Tool:** Vite 5
- **Styling:** Tailwind CSS 4
- **Charts & Graphs:** Recharts
- **Routing:** React Router 7
- **Icons:** React Icons
- **Utilities:** Day.js, UUID

## Prerequisites

Before you begin, ensure you have installed:
- Node.js (v18 or higher)
- npm (v9 or higher)
